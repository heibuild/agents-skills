# Bennis Agents and Skills

This repository contains my current setup for agentic coding tasks. Its a living repo and will change regularly.

## Skills

- **grill-me** my favorite one from Matt Pocock (source: https://github.com/mattpocock/skills)

- **to-prd** writes a plan into a nice PRD for agents to work on - added a switch to work either fully locally or with GitHub Issues (source: https://github.com/mattpocock/skills)

- **to-issues** nice skill to break bigger tasks / prds down to vertical features that can be implemented in parallel executions - added a switch whether to work fully locally or with GitHub Issues (source: https://github.com/mattpocock/skills)

- **playwright-screenshot** simple skill to let the AI "see" via screenshots a website. Super useful when building websites to have agent feedback.
